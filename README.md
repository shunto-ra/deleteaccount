# Sportish アカウントおよびデータ削除のリクエスト

Sportishをご利用いただきありがとうございます。
当アプリにおけるアカウント、および関連データの削除をご希望の場合は、以下のいずれかの方法で削除を行うことができます。

## 📱 1. アプリ内から削除する場合（推奨）
現在アプリをインストールしている場合は、アプリ内から即座にアカウントを削除できます。

1. Sportishアプリを開く
2. 自身の「プロフィール」画面を開く
3. 画面上の「三本線（メニュー）」をタップする
4. 「アカウント削除」を選択し、画面の指示に従って削除を完了する

## 📩 2. アプリをすでにアンインストールしている場合
すでにアプリを削除済みで、データの完全削除のみをご希望の場合は、メールにてリクエストを受け付けております。以下の内容をご送信ください。

* **送信先:** [あなたのサポート用メールアドレス]
* **件名:** Sportish アカウント削除申請
* **本文に記載していただく内容:**
  1. Sportishに登録しているメールアドレス

**【⏳ 処理にかかる期間】**
メールでのリクエストを受領してから、最大で7営業日以内にすべてのデータ（プロフィール、トレーニング記録など）の削除処理を完了し、完了のご報告をメールにて返信いたします。

---
**【⚠️ ご注意ください】**
一度削除されたデータは復元することができません。また、有料プランをご契約中の場合、アカウントを削除しても自動更新は解除されません。必ず事前にApp StoreまたはGoogle Playストアの設定から解約手続きを行ってください。
